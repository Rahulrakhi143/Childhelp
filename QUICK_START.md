## Installation & Setup

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Start Development Server**
   ```bash
   npm run dev
   ```

3. **Open Browser**
   - Navigate to `http://localhost:3000`
   - The app will automatically open in your default browser

## First Steps

1. **Create a Child Profile**
   - Click "Select Profile" or go to `/login`
   - Enter child's name
   - Choose an avatar
   - Click "Create Profile"

2. **Start Learning**
   - Click "Start Learning" on home page
   - Choose a language (Hindi, English, or Marwadi)
   - Select a subject
   - Pick a level (start with Alphabets)

3. **Practice**
   - **Writing Mode**: Draw letters/words on the canvas
   - **Speaking Mode**: Click "Record" and speak clearly
   - Follow hints and get feedback

4. **Check Progress**
   - Visit "Parent Panel" to see detailed reports
   - Track accuracy, streaks, and badges

## Browser Recommendations

- **Best Experience**: Chrome or Edge (for speech recognition)
- **Mobile/Tablet**: Works on all modern devices with touch support
- **Offline**: Works offline after initial load (data stored locally)

## Troubleshooting

**Speech Recognition Not Working?**
- Use Chrome or Edge browser
- Ensure microphone permissions are granted
- Check HTTPS connection (required in production)

**Handwriting Not Detecting?**
- Make sure you're drawing on the canvas
- Try clearing and drawing again
- Use tracing mode first for guidance

**Progress Not Saving?**
- Check browser local storage is enabled
- Clear browser cache if issues persist
- Data is stored locally (no server needed)
